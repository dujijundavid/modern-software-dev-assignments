# Technology Stack

## Backend
- **Framework**: FastAPI (Python 3.10+)
- **Database**: SQLite with SQLAlchemy 2.0+
- **Testing**: pytest + httpx (TestClient)

## Frontend
- Vanilla JavaScript + HTML/CSS

## LLM Integration
- **Local**: Ollama (llama3.1:8b model)
- **SDK**: OpenAI SDK (compatible with Ollama)

## Package Management
- **Poetry**: 2.2.1
- **Environment**: Conda (Python 3.12.7)

## Code Quality Tools
- **Formatter**: Black
- **Linter**: Ruff
- **Pre-commit**: Configured with hooks
