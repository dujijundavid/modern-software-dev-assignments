# META_LEARNING.md (v3)

> 目标：把“协作交付”升级为“可迁移、可复用、可运营”的工程能力。

**第一性原理**

* **能力**不是一次把事做出来，而是可复现的“决策与交付机制”。
* **学习**发生在真实任务中：假设 → 实验 → 反馈 → 修正（闭环）。
* **复利**来自把机制固化为资产：模板 / 自动化 / 评估 / 标准 / 平台。

---

## 0. 你在优化什么？（一句话北极星）

> 用最小的额外成本，把每次任务的有效机制沉淀为可复用资产，从而降低未来同类任务的边际成本。

## 1. 层级量表（Rubric）：证据指标 + 反证指标

> 规则：**只看证据，不看工具**。出现反证，默认降级处理。

### L1 工具使用者

**证据（满足 ≥3 条）**

* 能在项目里跑通：环境/命令/依赖/基本结果可复现。
* 能读改 AI 产出：修编译/类型/小缺陷，让结果可运行。
* 能提供足够上下文：文件路径、日志、复现步骤。

**反证（出现任一条）**

* 只能“复制粘贴试运气”，无法稳定复现。
* 不能解释改动原因，只能反复重试。

### L2 协作工程师

**证据（满足 ≥3 条）**

* 任务有清晰 **DoD（验收标准）**，不靠“感觉完成”。
* 有可执行 **验证计划**（测试/检查清单/回归步骤），并真的跑过。
* 能做方案权衡：复杂度/风险/维护成本，解释取舍。
* 能小步交付：把大任务切片，避免一次性大改。

**反证（出现任一条）**

* 无 DoD；或 DoD 不可验证。
* 不验证就合并；问题靠运气。

### L3 流程自动化者

**证据（必须同时满足 3 条）**

1. **可复用工件**：有清晰 I/O 契约 + 参数化 + 最小文档/示例。
2. **可运营**：至少有基础日志/错误信息可行动（能定位下一步）。
3. **复用收益真实**：复用 ≥3 次，且**每次人工介入 ≤5 分钟**。

**反证（出现任一条）**

* 每次运行都要“手工改 prompt/改路径/改配置”才能成功。
* 失败只能重跑，没有可诊断信息。
* 维护与调试成本 > 节省时间（净收益为负）。

### L4 系统架构师

**证据（必须同时满足 3 条）**

* 多个自动化可组合：接口/契约清晰，能编排协同。
* 有治理要素：观测（日志/指标）、权限/边界、版本/兼容、成本意识。
* 可被他人使用：**5 分钟上手**（安装→运行→看到结果）。

**反证（出现任一条）**

* 系统只能作者本人使用；换人即崩。
* 无版本/兼容策略，升级导致频繁破坏。

## 3. 资产账本（Asset Ledger）：把产出变资本

> 自动化不是目标；**净收益为正**的自动化才是资产。

### 3.1 资产分级（投资组合）

* **黄金资产**：高频（≥每周1次）、边界清晰、维护低。

  * 策略：优先投入，做到 L3 稳定复用。
* **灰色资产**：可能复用，但边界未清晰/需求变化大。

  * 策略：先做模板/半自动，等稳定后再系统化。
* **腐烂资产**：一次性、强耦合、外部依赖多、易过期。

  * 策略：坚决不系统化，走 L2 快速交付。

## 4. Eval Minimum：把“验证”升级为工程三分法

> L2/L3 默认使用；L3 自动化必须至少满足 1-1-1。

* **Spec Correctness（验收正确）**：是否满足 DoD。
* **Behavior Correctness（行为正确）**：边界/异常下是否稳定。
* **Regression Safety（回归安全）**：是否破坏既有能力。

### 4.1 L3 的最小 1-1-1

* 1 个 happy-path 示例（默认配置能跑通）
* 1 个 failure-path 示例（失败时给出可行动错误）
* 1 个回归断言（快照/黄金文件/最小测试均可）

## 5. 元技能（Meta Skills）：跨层发动机（可训练单元）

> 从 L2→L3→L4 的核心不是“写更多代码”，而是这些元技能更硬。

### 5.1 约束清单（Constraints)

**每个任务强制写 5 条约束**：数据/性能/安全/边界/依赖。

### 5.2 失败模式枚举（Mini-FMEA）

**每个 L3 资产至少列 5 个失败模式** + 对应信号 + 恢复动作。

### 5.3 基线与对照（Baseline）

自动化前先记录手工 baseline（频率×时间），否则 ROI 只是想象。

### 5.4 变更控制（Change Control）

每次重构必须给：回滚策略 + 版本标记 + 兼容说明（哪怕一句话）。

## 7. 两条关键纪律（避免系统变成负担）

1. **主文档必须可日用**：任何表单超过 3 分钟就要砍。
2. **只在黄金/灰色资产上追求 L3+**：腐烂资产直接走 L2 快交付。
